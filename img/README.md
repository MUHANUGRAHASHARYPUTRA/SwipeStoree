## Struktur Nama File Gambar (Panduan)

### Produk Umum
- Format: `p{produk_id}-{urutan}.jpg`
- Produk ID dimulai dari 1
- Urutan gambar dimulai dari 1
- Gunakan format JPG untuk kompatibilitas

### Produk Apple Style
- Format: `apple{urutan}.png`
- Urutan dimulai dari 1
- Gunakan format PNG untuk transparansi

## Resolusi Rekomendasi
- Gambar produk: 800x600px minimum, 1200x900px ideal
- Background hero: 1920x1080px minimum
- Logo: 300x100px
